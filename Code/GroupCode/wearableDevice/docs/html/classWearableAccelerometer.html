◆ addStepToCounter()

bool WearableAccelerometer::addStepToCounter ( )

Increments the step counter. If the it has been longer than the set period of time since the first step, then reset the time of the first step and reset the step counter. If not, and the step counter = 5 reset the step counter and time of first step, and return true.

Returns
true if 5 steps were detected within period of time, false otherwise.
